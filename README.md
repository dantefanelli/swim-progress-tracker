# Swim Progress Tracker

A Python application to track swim workouts, meet results, and performance goals.

## Features
- Log swim workouts (distance, time, type)
- Track meet results
- Set and monitor performance goals
- Generate summaries of progress

## Why I Built This
I created this project to combine my background in swimming with programming, and to start building real-world applications that track performance and data.

## Tech Used
- Python
